User Guide


Steps to follow:


  1. To learn algorithms using the tutorials, click on any one class of Algorithms of your interest in the side menu. For example: Clustering



  2. Choose one of the algorithm. For example: K-means Clustering in the Clustering class.



  3. Now you can see code and click on edit button to experiment over it. It will take you to google colab notebook hosted with github. So, you need to login with your respective gmail account to save any changes made to your local copy.






  4. If you want to contribute to our website. You can go to Home page and click on Contributions in the Navbar as shown in below picture and follow the instructions given in the contributions page.






  5. If you want to learn Data science from videos, we have resources section, which you can explore. Click on Resources in the home page and get started.
