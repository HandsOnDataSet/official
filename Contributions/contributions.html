How to Contribute ?


  1. Login with your Personal Github Account.
  2. Click on the following REPO LINK and FORK it.
  3. Make changes to the forked repository and Create Pull Request.

Steps to contribute in Contributions page


  1. Login with your Personal Github Account.
  2. Go to the following REPO LINK
  3. Click on the Contributions folder
  4. Click on the contributions.html file
  5. In the file contributions.html there is a div named card
  6. Copy the div named card and paste it again below the other divs
